# Obsidian Note Templates — Claude Projects + Ollama Imports

Use these templates when writing Claude Projects or Ollama pipeline output into the vault.
All imports land in `60-ClaudeProjects/imports/` as `status: draft` first.
File naming: `cp-[type]-[topic-in-kebab-case]-[YYYY-MM-DD].md`

---

## Processing Checklist

Run this after every import before changing status from `draft`:

```
- [ ] Front matter complete (title, date, tags, status: draft, source)
- [ ] Summary written in own words (not raw pipeline output)
- [ ] Key insights extracted as atomic bullet points
- [ ] Citations mapped to 30-Resources/ source notes
- [ ] Ollama tasks run and output reviewed (summarize / concepts / questions)
- [ ] Source filed in 30-Resources/ if it warrants a reference note
- [ ] Contradictions with vault notes flagged in ## Synthesis Notes
- [ ] [[wikilinks]] added to all recognized concepts
- [ ] Next actions listed: which insights to atomize, which MOCs to update
- [ ] File naming follows: cp-[type]-[topic]-[YYYY-MM-DD].md
```

Only change `status: draft` → `status: in-progress` once this checklist is complete.
Only change to `status: evergreen` once insights are atomized and fully linked.
